# 💬 Real-Time Chat App

A real-time multi-user chat application 
built with Node.js, Express, and Socket.io.

## 🚀 Features
- Multiple users can chat simultaneously
- Instant message delivery (no refresh needed)
- User join/leave notifications
- Responsive UI for mobile and desktop

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Real-time:** Socket.io

## 💡 How it works
Users connect to the server via Socket.io.
When one user sends a message, the server
instantly broadcasts it to all connected users.

## 📌 Example Use Cases
- Friends chatting in a group
- Team communication in real time
- Live customer support chat
- Classroom Q&A live session

## 🚀 How to Run
1. Clone this repository
2. Run `npm install`
3. Run `npm start`
4. Open browser → `localhost:3000`
5. Share link → Multiple users can chat!
